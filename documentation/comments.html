Comments

Returns an array of comments belonging to a thread.

Parameters
Name Type Description
select String Option specifying how the comments to be processed should be sorted
search String Option specifying whether the comments must contain a specific string
page Number Integer deciding specifying which sequence of comments should be returned, based on the "paging" system - 5 comments per page.
thread-id String Identifier for the thread to be fetched


  • curl -X GET ./comments?select=date-newest&search=&page=1/

Response fields
Name Type Description
Array Array An array displaying the five comments
Index zero [0] String The unique ID given for each comment
Index one [1] String The text body of each comment
Index two [2] String The date the comment was posted, in string format
Index three [3] Number The number of likes the thread has
Index four [4] String The parent thread ID attributed to the comment
Index five [5] String A reference to the image included in the comment
