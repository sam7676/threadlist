Thread info

Returns information on a single thread.

Parameters
Name Type Description
id string The ID of the given thread


  • curl -X GET .threadinfo?id=00000004/


{
"id": "00000004"
"title": "Thread title"
"body": "Thread body"
"date":"2023-12-27-00-27-30"
"lastupdate": "2023-12-27-00-27-37"
"likes": 1
"comments": 0
}

Response fields
Name Type Description
id String The unique ID given for each thread
title String The title of each thread
body String The body for each thread
date String The date the thread was posted, in string format
lastupdate String The date the thread was last updated, in string format
likes Number The number of likes the thread has
comments Number The number of comments the thread has
