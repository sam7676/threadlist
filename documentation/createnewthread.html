Create new thread

Adds a thread to the database.
This includes a title and a body of text.

Parameters
Name Type Description
title String Title of the thread
body String Body of the thread


  • curl -X POST ./createnewthread/


{
"error" : "length"
}

Response fields
Name Type Description
Error String Sends a message to the user that the length of the thread/body isn't in the required range.
