Threads

Returns five threads to be displayed, satisfying the given search conditions.

Parameters
Name Type Description
select string The chosen method to sort the threads by
search string Returns only threads that contain the given string in their title
page number Selects five threads based on their index in the sorted array


  • curl -X GET .threads?select=date-newest&search=&page=1/

Name Type Description
Array Array An array displaying the five threads

Index one [1] String The title of each thread
Index two [2] String The body for each thread
Index three [3] String The date the thread was posted, in string format
Index four [4] String The date the thread was last updated, in string format
Index five [5] Number The number of likes the thread has
Index six [6] Number The number of comments the thread has
