Get comment count

Returns the number of pages of comments there are for a given thread.
A page is defined to be a bundle of 5 threads.

Parameters
Name Type Description
thread-id String ID of the thread to get the comment count of


  • curl -X GET ./getcommentcount?thread-id=00000004/


{
"page_count" : 1
"comment_count" : 4
}

Response fields
Name Type Description
page_count Number Returns the number of pages available to the user.
comment_count Number Returns the total number of comments attributed to a thread.
