Add comment

Adds a comment to the database.
This includes a body of text or an image.

Parameters
Name Type Description
Form data FormData object Object including thread ID, image encoded as FormData, and a form text body


  • curl -X POST ./addcomment/


{
"error" : "content"
}

Response fields
Name Type Description
Error String Sends a message to the user that the content specified wasn't sufficient to post a comment. This is opposed to a 404 error, as the server has been communicated with.
