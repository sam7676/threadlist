Get last update

Returns a date specifying the last time a thread was updated.

Parameters
Name Type Description
thread-id String ID of the thread to get the last update time of


  • curl -X GET ./getlastupdate?thread-id=00000004/


{
"last_update" : "2023-12-27-00-27-30"
}

Response fields
Name Type Description
last_update String A string representing the time the last time the given thread was updated.
